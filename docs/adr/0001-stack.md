# ADR 0001: Stack Base

## Decisão

O projeto adota:

- `Expo`
- `React Native`
- `TypeScript`
- `zod` para validação de contratos
- `Jest` com preset Expo para testes automatizados

## Justificativa

Essa combinação favorece iteração rápida no mobile, organização clara entre interface e domínio, e uma base simples para evoluir o produto sem inflar a complexidade logo no MVP.
